# 云端部署指南 — 像问卷星一样随时随地使用

## 为什么需要云端部署？

局域网方式需要手机和电脑在同一WiFi，不够方便。
**云端部署后，患者扫码即可填写，不受网络限制，和问卷星体验完全一致。**

---

## 方案一：Render.com 一键部署（推荐，免费）

Render 是国际知名的云平台，支持免费部署 Python 应用，中国可访问。

### 步骤

#### 1. 注册 Render 账号
打开 https://render.com → 点击 "Get Started" → 用 GitHub 账号注册（免费）

#### 2. 上传代码到 GitHub
```bash
# 在 GitHub 上创建一个新仓库（如 qlq-survey）
# 然后将量表文件夹中的文件上传到仓库：
#   Dockerfile, render.yaml, requirements.txt
#   survey_app.py, QLQ-Combined-Survey.html
```

或者我可以帮您上传。

#### 3. 一键部署
在 Render Dashboard：
- 点击 "New" → "Blueprint"
- 选择刚才创建的 GitHub 仓库
- Render 自动读取 `render.yaml` 完成部署
- 部署成功后获得地址如：`https://qlq-survey.onrender.com`

#### 4. 设置域名
部署成功后：
- 在 Render Dashboard 找到服务
- 进入 "Environment" 标签
- 添加环境变量：`BASE_URL` = `https://你的地址.onrender.com`
- 点击 "Save Changes"，服务自动重启

#### 5. 生成二维码
- 重启后二维码会自动生成（使用 `BASE_URL`）
- 也可以手动访问 `https://你的地址.onrender.com/` → 终端会打印二维码
- 或者用任意在线二维码生成器：把问卷地址粘贴进去即可

#### 6. 使用
- 📱 患者扫码 → 填写问卷 → 提交
- 📊 医生打开 `https://你的地址.onrender.com/admin` → 查看统计、导出数据

## 方案二：Railway 部署（更稳定，免费额度）

Railway 国内访问速度更快。

### 步骤
1. 打开 https://railway.app → 用 GitHub 登录
2. 点击 "New Project" → "Deploy from GitHub Repo"
3. 选择仓库，Railway 自动检测 Dockerfile
4. 部署完成后设置环境变量 `BASE_URL`
5. 获得地址如 `https://qlq-survey.up.railway.app`

---

## 方案三：阿里云/腾讯云（国内最快，需付费）

如果有阿里云或腾讯云账号（约 ¥50-100/月）：

```bash
# 1. 购买轻量应用服务器（最低配置即可）
# 2. SSH 登录后运行：
pip3 install flask "qrcode[pil]"
# 3. 上传文件并启动：
nohup python3 survey_app.py > survey.log 2>&1 &
# 4. 在云控制台开放 8080 端口
# 5. 获得公网 IP，访问 http://IP:8080/
```

## 方案四：Cloudflare Tunnel（免费，需要 Mac 一直开机）

如果不想注册新账号，可以让 Mac 长期开着：

```bash
# 1. 安装 cloudflared
brew install cloudflared

# 2. 启动（需要免费 Cloudflare 账号）
cloudflared tunnel --url http://localhost:8080

# 3. 获得公网地址如 https://xxx.trycloudflare.com
```

## 部署后的使用

无论哪种方案，部署成功后的使用方式完全相同：

| 角色 | 操作 |
|------|------|
| **患者** | 扫描二维码 → 手机浏览器打开问卷 → 填写 → 提交 |
| **医生** | 浏览器打开 `管理后台地址` → 查看统计 / 浏览记录 / 导出 CSV |

## 常见问题

**Q: 免费方案够用吗？**
A: Render 免费套餐每月 750 小时（够 24/7 运行），适合日均几十份问卷的场景。

**Q: 数据安全吗？**
A: 所有数据存储在平台提供的持久化磁盘上。建议定期在管理后台导出 CSV 备份。

**Q: 可以设置密码保护管理后台吗？**
A: 可以。我可以为管理后台添加简单密码验证。

**Q: 二维码在哪？**
A: 服务器启动时自动生成 `survey_qrcode.png`，也在终端直接打印。也可以用任何二维码生成网站把问卷 URL 转成二维码。
